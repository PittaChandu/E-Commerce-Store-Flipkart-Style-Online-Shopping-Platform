# 🛒 E-Commerce Store - Flipkart Style Web Application

## 🌍 Live Demo
👉 https://pittachandu.github.io/E-Commerce-Store-Flipkart-Style-Online-Shopping-Platform/

This is a Flipkart-style E-Commerce web application developed as a team project to simulate a real-time online shopping platform. The website allows users to browse products, add items to cart, and manage their shopping experience through an interactive and responsive interface.

## 🚀 Features
- Dynamic product listing
- Add to cart functionality
- Cart total calculation
- Login simulation
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run Locally

1. Clone the repository:
   git clone https://github.com/PittaChandu/E-Commerce-Store-Flipkart-Style-Online-Shopping-Platform.git

2. Navigate to the project directory:
   cd E-Commerce-Store-Flipkart-Style-Online-Shopping-Platform

3. Open the project in VS Code.

4. Run the index.html file in your browser.
